M4: Lost Relic Game Jam

Unity
June 19th, 2022

During a game jam, everyone makes and submits a game that is linked to the same theme. I joined the Lost Relic game jam. The theme was "connection", so I immediately started thinking. I already made some games in the past and i was not planning to make a very detailed and prize-winning game for this jam. I mostly joined this jam because I wanted to explore the community and see how they planned and excecuted this challenge.

This is the game I submitted. In this game you can control two rectuangles, but they are connected to eachother with an unbreakable line. The player can however change the length of this line by collecting hearts and using the scrollwheel. You can play it here.
