M2: Programming

Arduino, Processing, Wiring, Teamwork
Feb 26th, 2022

This module we learned about Arduino. Arduino is really cool and we learned quite a few things in the Programming course this module. For the final assignent of this course you had to make an Arduino and Processing system. I worked with Natan, and we decided to make a game controller on the Arduino so you could play a game in Processing. It was a really fun project and it turned out great!
We made it so you could control a plane using motion sensors and a joystick. In the processing part, you would have to navigate the plane trough rings to score points. The arduino also makes sounds and it shows your current lives.
